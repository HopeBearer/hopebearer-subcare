# SubCare Web App

A modern subscription management frontend built with Next.js, TypeScript, and UnoCSS.

## Features

- **Authentication**: Secure Login and Registration flow with JWT support.
- **Dashboard**: Protected dashboard view for subscription overview.
- **UI System**: Custom UI components built with UnoCSS and Tailwind Merge.
- **State Management**: Client-side auth state managed via Zustand.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: UnoCSS (Atomic CSS) + Sass
- **State**: Zustand
- **Forms**: React Hook Form
- **API Client**: Axios

## Getting Started

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Run development server:
   ```bash
   pnpm dev --filter web
   ```
